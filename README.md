# JPMC-5
JPMC-5

Hello! This project aimed to develop a ML model to accurately predict home value for homes worth more than $1M and identify features/correlations that influence these high-value properties.

Our objectives: 
- Develop a regression ML model to predict home values for $1M+ Homes
- Identify important features/correlations that influence $1M+ Homes

Our outcome: A regression model that provides precise home value predictions for $1M+ homes along with details on important features and their relationships to home values, offering valuable insights for a real estate investor.
